## ⚙️ How to Use

1. Clone the repository  
   ```bash
   git clone 'https://github.com/Gauravgit97/LeetCode.git'
   ```
2. Navigate to the project folder  
   ```bash
   cd LeetCode
   ```
3. Open any folder (e.g., Arrays) and run a Python file  
   ```bash
   python arrays/twoSum.py
   ```
